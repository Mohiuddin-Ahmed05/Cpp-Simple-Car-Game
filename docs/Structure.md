# Game (SDL2 Starter)

A minimal SDL2 + SDL2_ttf starter project with a simple **scene system**:
- **Menu** (Start / Options / Quit)
- **Play** (player movement demo)
- **Options** (fullscreen + resolution toggle)

---

## Folder Structure

```
/Game
  CMakeLists.txt

  assets/
    fonts/
      DejaVuSans.ttf

  build/                 # generated by CMake (out-of-source build)

  docs/                  # optional project docs

  src/
    main.cpp
    Game.h
    Game.cpp
    Scene.h

    MenuScene.h
    MenuScene.cpp

    PlayScene.h
    PlayScene.cpp

    OptionsScene.h
    OptionsScene.cpp

    Text.h
    Text.cpp
```

---

## Build & Run

From the project root (`/Game`):

```bash
cmake -S . -B build
cmake --build build
./build/game
```

---

## Controls

### Global
- **ESC**
  - From **Menu**: Quit
  - From any other scene: Return to **Menu**

### Menu Scene
- **W / Up Arrow**: move selection up
- **S / Down Arrow**: move selection down
- **Enter**: confirm selection

### Play Scene
- **WASD / Arrow Keys**: move player square

### Options Scene
- **F**: toggle fullscreen (desktop fullscreen)
- **R**: cycle windowed resolution (only when not fullscreen)

## File Overview

### `CMakeLists.txt`
Build configuration using `pkg-config` to locate **SDL2** and **SDL2_ttf** and compile all source files.

---

### `src/main.cpp`
Application entry point:
- Initializes SDL2 and SDL_ttf
- Creates window and renderer
- Loads font assets
- Runs the main game loop via `Game`
- Cleans up on exit

---

### `src/Scene.h`
Abstract base class for all scenes:
- `handleEvent`
- `update`
- `render`

---

### `src/Game.h / src/Game.cpp`
Central game controller:
- Owns main loop and timing
- Manages active scene
- Handles global input (ESC)
- Applies fullscreen and resolution changes safely

---

### `src/MenuScene.*`
Main menu implementation:
- Start / Options / Quit
- Keyboard navigation
- Animated selection highlight

---

### `src/PlayScene.*`
Gameplay demo:
- Player movement
- Diagonal normalization
- Window boundary clamping

---

### `src/OptionsScene.*`
Display options:
- Fullscreen toggle
- Windowed resolution cycling
- Current mode display

---

### `src/Text.*`
Reusable text rendering utility:
- Centers UTF-8 text inside a rectangle using SDL_ttf

---

### Assets
- `assets/fonts/DejaVuSans.ttf`: font used across UI

---

## Notes
This structure is intentionally minimal but scalable. New scenes and systems can be added without refactoring core files.
